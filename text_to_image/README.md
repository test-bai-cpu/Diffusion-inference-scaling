### Elucidating the design space of BFS

The repo is for experiments in text to image generation with BFS, and is adapted from [FK-Diffusion-Steering](https://github.com/zacharyhorvitz/Fk-Diffusion-Steering). To run the experiments, run `launch.sh` and experiment with different `--potential_type`, `--resampling`, `--tempering_schedule` options.
